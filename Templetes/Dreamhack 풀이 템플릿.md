## 코드
---
👉 “공격 전략을 구현한 결과물”

### 기준

- 불필요한 코드 제거
- 공격 핵심만 남김
- 변수명은 수식과 대응되게

### crypto 기준 팁

- print 중간 결과 넣어도 OK
- 수식과 코드가 매칭되게 작성

### 예시

`from Crypto.Util.number import *  p = GCD(n1, n2) q = n1 // p phi = (p-1)*(q-1) d = inverse(e, phi) m = pow(c, d, n)`
