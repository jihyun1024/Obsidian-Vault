# 지니 계수(Gini Coefficient)란?
지니 계수는 **소득, 부의 불평등 정도를 수치화한 지표**로, 0에서 1 사이의 값을 가진다. 

- 0: 완전 평등 (모두 부자거나, 모두 가난하거나)
- 1: 완전 불평등 (한 사람이 모든 소득 차지)

---

# 로렌츠 곡선을 사용한 정의
**기본 정의**
$$G = \frac{A}{A \ +\ B}$$
- $A$: 완전평등선과 로렌츠 곡선 사이의 면적
- $B$: 로렌츠 곡선 아래의 면적

이를 적분으로 표현하면 다음과 같다. 
$$G=1−2\int_{0}^{1}​L(p)dp$$
$L(p)$: 하위 $p$%가 차지하는 누적 소득 비율

---

# 실제 소득 데이터를 사용한 지니 계수 계산식
## 정의에 가장 충실한 공식
소득이 $x_1,\ x_2,\ ...\ ,\ x_n$일 때, 
$$G = \frac{\sum_{i=1}^n\sum_{j=1}^n|x_i - x_j|}{2n^2\hat{x}}$$
- $n$: 전체 인원
- $\hat{x}$: 평균 소득

## 가장 실용적인 단일합 공식 (정렬된 데이터 기반)
소득이 오름차순 정렬되어 있을 때, 
$$G=\frac{2\sum_{i=1}^{n}ix_i}{n\sum_{i=1}^{n}x_i} - \frac{n+1}{n}$$
- 통계, 데이터 분석에서 가장 많이 쓰이는 공식
- 복잡해 보이지만, 계산량이 적어 실전에서 많이 쓰인다. 

---

# 지니 계수 해석 기준 (일반인 기준)
- 0.20 이하: 매우 평등
- 0.20 ~ 0.35: 비교적 평등
- 0.35 ~ 0.50: 불평등 심화
- 0.50 이상: 불평등 매우 심함

---

# Python 계산 코드
```python
def gini(x):
	x = sorted(x)
	n = len(x)
	cumulative = sum((i + 1) * x_i for i, x_i in enumerate(x))
	total = sum(x)
	return ((2 * cumulative) / (n * total)) - ((n + 1) / n)
	
print(gini([10, 20, 30, 40, 100]))
```

코드를 보면 알겠지만, 정렬된 데이터 기반 계산 공식을 사용한 것이다.
