
|명령|설명|사용 예|
|---|---|---|
|and|논리곱(AND)|and eax, 0|
|or|논리합(OR)|or ebx, 0xffffffff|
|xor|배타적 논리합(XOR)|xor ebx, 0x0000ffff|
|not|오퍼랜드에 1의 보수를 취함 (비트 반전)|not ebx|
|shl|왼쪽으로 오퍼랜드만큼 이동, 최하위 비트는 0으로 채워짐|shl eax, 1|
|shr|오른쪽으로 오퍼랜드만큼 이동, 최상위 비트는 부호비트로 채워짐|shr eax, 2|
|rol / rcl|왼쪽으로 오퍼랜드만큼 순환|rol eax, 1|
|ror / rcr|오른쪽으로 오퍼랜드만큼 순환|ror eax, 2|

비트 연산자는 비트 단위로 연산을 수행하며, 논리합, 논리곱, 배타적 논리합, 비트 반전 등의 연산이 이에 해당한다. 

또한, 이동 연산은 곱셈과 나눗셈 연산을 구현할 수 있는데, 아래 코드는 비트 이동으로 곱셈을 수행한다. 

| 명령어        | 설명                 |
| ---------- | ------------------ |
| shl eax, 2 | eax에 4를 곱한 결과와 같다. |
| shr eax, 3 | eax에 8을 곱한 결과와 같다. |
