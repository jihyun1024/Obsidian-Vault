프로그램의 흐름 전환을 결정하는 명렁어로, 사용 예시는 다음과 같다. 

- jmp address → EIP 레지스터에 해당 주소를 무조건 넣음
- jcc address → cc 조건에 따라 EIP 레지스터에 해당 주소를 넣음

|조건(cc)|설명|플래그값|
|---|---|---|
|E / Z|같은 / 0인 (**Equal**)|ZF = 1|
|NE / NZ|다른 / 0이 아닌 (**Not Equal**)|ZF = 0|
|L / NGE|작은 (부호 있는 연산, **Less / Not Greater or Equal**)|(SF^0F) = 1|
|G / NLE|큰 (부호 있는 연산, **Greater / Not Less or Equal**)|((SF^0F)|
|GE / NL|크거나 같은 (부호 있는 연산, **Greater or Equal / Not Less**)|(SF^0F) = 0|
|LE / NG|작거나 같은 (부호 있는 연산, **Less or Equal / Not Greater**)|((SF^0F)|
|B / NAE|작은 (부호 없는 연산, Branch / **Not Above or Equal**)|CF = 1|
|AE / NB|크거나 같은 (부호 없는 연산, **Above or Equal / Not Branch**)|CF = 0|
|P / PE|PF가 1이면 (짝수, **Parity / Parity Even**)|PF = 1|
|NP / P0|PF가 0이면 (홀수, **Not Parity / Parity Odd**)|PF = 0|
|0|OF가 1이면 (오버플로우가 일어나면, **Overflow**)|0F = 1|
|S|SF가 1이면 (연산 결과가 음수이면, **Signed**)|SF = 1|
|ECXZ|ECX가 0이면 (카운터가 0이면, **Enhanced Clear and eXit if Zero**)|ECX = 0|
