통합 자원 식별자(Universal Resource Identifier, URI)는 인터넷에 있는 자원을 나타내는 유일한 주소이다. 
URI의 존재는 인터넷에서 요구되는 기본 조건으로 인터넷 프로토콜에 항상 붙어 다닌다. 
URI의 하위 개념으로 URL, URN이 있다. 
### 표현방법
일반 URI와 절대 URI 참조 문법은 [RFC 2396](https://www.ietf.org/rfc/rfc2396.txt)에 처음 정의되었으며, [RFC 3986](https://datatracker.ietf.org/doc/html/rfc3986)으로 완성되어 2005년 1월 출판되었다. 
일반 URI는 다음과 같은 형태를 나타낸다. 
`scheme:[//[user[:password]@]host[:port]][/path][?query][#fragment]`
