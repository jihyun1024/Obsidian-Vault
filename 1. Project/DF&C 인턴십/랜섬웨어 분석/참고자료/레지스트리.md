레지스트리란 윈도우 운영 체제의 설정, 선택 항목을 담고 있는 데이터베이스로, 모든 하드웨어, OS 소프트웨어, 응용 프로그램, 사용자 PC 선호도 등의 정보와 설정 값이 들어 있다. 
사용자가 제어판 설정, 파일 연결, 시스템 정책, 설치된 소프트웨어 등을 변경하면 이에 따른 변경 사항들이 레지스트리에 반영되어 저장된다. 

레지스트리는 키와 값이라는 두 가지의 기본 요소를 담고 있다. 
레지스트리 키는 폴더와 비슷하며, 각 키는 수많은 서브키를 가질 수 있는 서브키를 포함할 수 있다. 
레지스트리 값은 키 안에 들어 있는 이름/자료로, 값은 여러 키로부터 따로 참조할 수 있다. 

--- 
레지스트리는 Windows - 실행 - regedit.exe로 실행가능하다. 
![[Pasted image 20250723151830.png]]
1. HKEY_CLASSES_ROOT: 관리자 계정 관련 확장자 설정, 기타 프로그램 설정 등
2. HKEY_CURRENT_USER: 사용자별 확장자 설정, 기타 프로그램 설정
3. HKEY_LOCAL_MACHINE: 모든 사용자의 설정, 시스템 정보
4. HKEY_USERS: 각각의 사용자 프로파일에 대한 HKEY_CURRENT_USER 키에 일치하는 서브키
5. HKEY_CURRENT_CONFIG: 실행 시간에 수집한 자료들로, 영구적으로 저장되지는 않음. 

레지스트리 정보가 저장된 파일들은 C:\Windows\System32\config 폴더에서 찾을 수 있으며, 이런 파일들을 하이브라고 한다. 레지스트리 편집기에서 하이브 파일을 편집기로 가져와 편집할 수 있다. 
1. SOFTWARE: HKEY_LOCAL_MACHINE\SOFTWARE 내에 존재하며, 윈도우 및 소프트웨어 전역 설정 정보를 담고 있음
2. SYSTEM: HKEY_LOCAL_MACHINE\SYSTEM 내에 존재하며, 시스템 전역 설정 정보를 담고 있음
3. SECURITY: HKEY_LOCAL_MACHINE\SECURITY 내에 존재하며, 권한, 인증 도메인 등의 보안 관련 정보를 담고 있음
4. SAM: HKEY_LOCAL_MACHINE\SAM 내에 존재하며, 사용자 별 암호화된 비밀번호 저장
5. (USERNAME)\NTUSER.DAT: HKEY_CURRENT_USER 내에 존재하며, 각 윈도우 계정의 HKEY_CURRENT_USER 키를 담고 있음
