### 다양한 타입의 비밀 정보들을 보관 가능
![[Pasted image 20250710003703.png]]
밑에 있는 탭들 중 [타입] 탭에 들어가면 다양한 사용자의 정보들을 보관할 수 있었음

1. 로그인: 웹에서 사용자의 아이디와 비밀번호를 저장
2. 은행: 사용자의 계좌번호와 예금주 성명 저장
3. 카드: 사용자의 카드 일련번호와 카드 CVC 저장
4. 신분증: 주민등록증, 운전면허증, 여권 정보 저장
5. 바코드: 바코드 밑의 번호 저장
6. 이미지: 사용자가 중요하다고 판단하는 이미지 저장

### 정보들을 그룹화하여 보관 가능
![[Pasted image 20250710004302.png]]
정보들을 연관성 있는 정보끼리 그룹화하여 자신만 보거나, 다른 사람에게 공유할 수 있는 공유 그룹의 형태로 저장
공유 그룹의 경우, 공유할 계정의 계정 코드를 복사하여 공유할 계정을 추가 가능 (계정 코드: 앱 설정-계정)

### 2단계 인증
* 앱을 실행 후 2단계 인증 탭으로 이동
* 추가 버튼을 눌러 QR 코드 스캔 옵션을 선택하여 카메라 접근 권한을 허용
* 카메라로 QR코드를 스캔하면 자동으로 계정 정보를 인식하여 등록

### 패스워드 랜덤 생성 기능
![[Pasted image 20250710005053.png]]
* 앱을 실행 후 암호생성 탭으로 이동
* 길이, 특수문자 포함 여부, 숫자 포함 여부, 대문자 포함 여부 등을 전부 선택하여 비밀번호를 생성
해당 비밀번호를 등록할 경우, 로그인 정보로 인식되어 아이디만 생성하면 상대적으로 안전한 로그인 정보를 생성 

### 다양한 부가 기능
1. 사용자 지정 필드 추가
	* 사용자 데이터를 저장할 때 '사용자 지정 필드 추가'를 선택
	* 웹 사이트 정보, 메모, 주소 등 다양한 부가 정보들을 저장 가능
2. 앱 잠금
	* 설정에 들어가 앱 잠금을 선택
	* 4자리 PIN이나 8자리 이상의 비밀번호를 설정하여 앱 실행 시 잠금해제
	* 생체인증을 추가로 등록하여 PIN이나 비밀번호 대신 생체인증으로 잠금해제 가능
3. 민감한 정보 숨기기
	* 설정-보안에 들어가 민감한 정보 숨기기를 선택
	* 평소에는 아이디와 비밀번호가 보이지 않다 눈 모양 아이콘을 클릭하여 원하는 정보만 확인
4. 데이터 가져오기 or 내보내기
	* 데이터 가져오기: .csv 형식 템플릿 다운로드 - 형식에 맞게 데이터를 입력 - 데이터 가져오기에 등록 - 첫 화면으로 이동하여 데이터 확인
	* 데이터 내보내기: 저장된 모든 데이터를 .xlsx 파일 형식으로 내보냄
5. 리뉴얼 이전의 데이터 복원
