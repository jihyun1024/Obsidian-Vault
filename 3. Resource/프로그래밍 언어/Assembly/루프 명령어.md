### rep 명령어

ECX 레지스터를 이용해 ECX 레지스터에 지정된 횟수 또는 ZF 플래그의 조건이 충족되지 않을 때까지 문자열 명령어를 반복한다. 

- REP 명령어: INS, OUTS, MOVS, LODS, STOS 명령어에 추가 가능
- 나머지 명령어: CMPS, SCAS 명령어에 추가 가능

|REP 명령|설명|
|---|---|
|rep|ECX가 0이 될 때까지 반복|
|repe / repz|ECX가 0이거나 ZF가 0일 때까지 반복|
|repne / repnz|ECX가 0이거나 ZF가 1일 때까지 반복|

- N은 b(8비트), w(16비트), d(32비트), q(64비트)가 사용된다

|REP 관련 명령|설명|
|---|---|
|insN|DX 레지스터가 지정한 I/O 포트에서 ESI 또는 RSI가 지정한 메모리로 N바이트 데이터를 입력|
|outsN|ESI 또는 RSI가 저장한 메모리에서 DX 레지스터가 지정한 I/O 포트로 N바이트 데이터를 출력|
|movsN|ESI 또는 RSI 주소에서 EDI 또는 RDI 주소로 N바이트 데이터를 이동, memcpy()와 동일|
|lodsN|ESI 또는 RSI 주소에서 N바이트 데이터를 EAX 또는 RAX 레지스터로 적재|
|cmpsN|ESI 또는 RSI 주소의 값과 EDI 또는 RDI 주소의 값을 N바이트 비교, memcmp()와 동일|
|scasN|EDI 또는 ESI 주소의 값과 EAX 또는 RAX 레지스터 값을 N바이트 비교|
|stosN|EDI 또는 ESI 주소의 위치에 EAX 또는 RAX 레지스터 값을 저장, memset()과 동일|

### loop 명령어

|LOOP 명령|설명|
|---|---|
|loop rel|RCX 또는 ECX 카운터를 감소, 카운터가 0이 아니면 rel 주소로 이동|
|loope rel|RCX 또는 ECX 카운터를 감소, 카운터가 0이 아니고 ZF가 1이면 rel 주소로 이동|
|loopne rel|RCX 또는 ECX 카운터를 감소, 카운터가 0이 아니고 ZF가 0이면 rel 주소로 이동|
관계된 위치까지를 ecx, 또는 rcx 레지스터 크기만큼 반복 수행한다.
