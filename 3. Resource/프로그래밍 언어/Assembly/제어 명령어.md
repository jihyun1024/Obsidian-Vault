프로그램의 흐름을 제어하기 위해 필요한 어셈블리어 명령어로, 두 개의 값을 비교하고 그 결과에 따라 다른 루틴을 수행하는 명령어이다. 
cmp의 경우, 오퍼랜드의 종류나 크기에 따라 cmps, cmppd, cmpsd 등의 다양한 명령어가 존재한다.

| 명령    | 설명                             | 사용 예            |
| ----- | ------------------------------ | --------------- |
| cmp   | 두 개의 오퍼랜드끼리 뺄셈 연산을 하여 플래그를 설정  | cmp ecx, 0      |
| test  | 두 개의 오퍼랜드끼리 AND 연산을 하여 플래그를 설정 | test eax, eax   |
| call  | 해당되는 주소의 함수를 호출                | call 0x12345678 |
| int   | 0xcd: 오퍼랜드로 지정된 예외 처리 수행       | int 3           |
|       | 0xcc: 중단점으로 디버그용 소프트웨어 트랩      | int3            |
|       | 0xce: 오버플로우 예외 상황시 발생하는 트랩     | into            |
|       | 0xf1: 하드웨어 디버그 트랩              | int1            |
| leave | 함수에서 사용한 지역변수 스택 초기화           | leave           |
| ret   | 스택에 저장된 주소로 복귀                 | ret             |
| nop   | 아무 동작도 수행하지 않음                 | nop             |

추가로, 함수의 끝을 나타내는 과정에서 leave와 ret을 사용한다.
